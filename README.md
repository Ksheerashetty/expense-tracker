# Expense Tracker

A simple React-based expense tracker app to manage daily spending.

## Features
- Add new expenses
- View recent transactions
- Track total spending

## Tech Stack
- React (with vite)
- JavaScript
- CSS
- Chart.js

## How to Run
1. Clone the repo
2. Run `npm install`
3. Run `npm start`
